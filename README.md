# DolCon - Dominion of Light

A console-based RPG where players explore a procedurally generated world, navigate between cells and settlements, engage in turn-based combat, and manage inventory and equipment.

## Requirements

- .NET 10.0 SDK

## Running

```bash
# Console version
dotnet run --project DolCon/DolCon.csproj

# MonoGame version
dotnet run --project DolCon.MonoGame/DolCon.MonoGame.csproj
```

## Testing

```bash
dotnet test
```
